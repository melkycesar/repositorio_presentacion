# Repositorio de Presentación — Nortica Systems

Repositorio con los recursos visuales, documentos comerciales/técnicos y elementos de identidad gráfica (UI) utilizados en las presentaciones y material corporativo de **Nortica Systems**.

## Estructura del repositorio

```
repositorio_presentacion/
├── imagenes_contenido_presentacion/   # Contenido visual y documental de las presentaciones
│   ├── Accesorios/                    # Imágenes de equipos y accesorios industriales (bombas, turbinas, transformadores, etc.)
│   ├── Brochure/                      # Páginas del brochure corporativo (8 páginas, formato imagen)
│   ├── Certificado/                   # Certificado de la empresa
│   ├── Commercial Proposal/           # Páginas de la propuesta comercial (3 páginas)
│   ├── Commissioning Checklist/       # Checklist de pre-comisionamiento (2 páginas)
│   ├── Ejemplos uso Nueva Linea Grafica/  # Ejemplos de aplicación de la nueva línea gráfica (portada, paleta de color, tipografías, presentaciones/web, accesorios)
│   ├── Factura Hotel/                 # Factura de hotel (documento de respaldo)
│   ├── Factura Soporte/               # Factura de soporte (documento de respaldo)
│   ├── Factura Stimated Quotation/    # Factura de cotización estimada (2 páginas)
│   ├── Factura para Credito Fiscal/   # Factura para crédito fiscal
│   ├── Fondos Presentaciones/         # Fondos/plantillas para presentaciones (21 imágenes numeradas)
│   ├── Iconos/                        # Set de íconos temáticos (energía, automatización, equipos, personas, etc.)
│   ├── Manual Presentaciones/         # Manual de uso de presentaciones (10 páginas)
│   ├── Presentacion/                  # Presentación corporativa completa (23 páginas)
│   ├── Service Report/                # Reporte de servicio (8 páginas)
│   ├── Technical Data/                # Hoja de datos técnicos (11 páginas)
│   └── Test/                          # Material de prueba/test (13 páginas)
│
├── ui_presentacion/                   # Elementos de interfaz e identidad visual
│   ├── logos_nortica/                 # Logotipos oficiales de Nortica Systems
│   │   ├── logo_blanco_para_fondo_orcuro_un_color.png   # Logo en blanco, un color, para fondo oscuro
│   │   ├── logo_blanco_y_cyan_para_fondo_orcuro.png     # Logo en blanco y cyan, para fondo oscuro
│   │   ├── logo_default.png                             # Logo por defecto (uso general)
│   │   └── logo_manual.png                               # Logo usado en el manual de marca
│   └── imagen_fondo_ui/               # Imagen de fondo para interfaces/herramientas internas
│       └── background_image.png
│
└── firma_correos/                     # Plantilla de firma de correo electrónico corporativa
    ├── firma.html                     # Código HTML de la firma de correo
    └── images/                        # Imágenes embebidas en la firma (logo e imagen de firma)
        ├── img_signature.png
        └── logo_default.png
```

## Descripción de carpetas

### `imagenes_contenido_presentacion/`
Contiene todo el contenido gráfico y documental que alimenta las presentaciones y materiales de Nortica Systems, organizado por tipo de documento:

| Carpeta | Contenido |
|---|---|
| `Accesorios/` | Imágenes individuales de equipos industriales (bombas, turbinas, transformadores, generadores, válvulas, consolas SCADA, etc.) usadas como recursos gráficos. |
| `Brochure/` | Brochure corporativo de la empresa, páginas exportadas como imagen (`.jpg`). |
| `Certificado/` | Certificado(s) oficiales de la empresa. |
| `Commercial Proposal/` | Propuesta comercial exportada como imágenes. |
| `Commissioning Checklist/` | Checklist de pre-comisionamiento de equipos. |
| `Ejemplos uso Nueva Linea Grafica/` | Ejemplos visuales de cómo aplicar la nueva línea gráfica de la marca (portada, paleta cromática, tipografías, presentaciones/página web, accesorios). |
| `Factura Hotel/`, `Factura Soporte/`, `Factura Stimated Quotation/`, `Factura para Credito Fiscal/` | Facturas y documentos administrativos/comerciales de respaldo. |
| `Fondos Presentaciones/` | Set de 21 fondos/plantillas base numeradas para construir presentaciones. |
| `Iconos/` | Íconos temáticos individuales (válvulas, turbinas, subestaciones, HMI, DCS, equipos, personas, etc.) para uso en presentaciones y documentos. |
| `Manual Presentaciones/` | Manual con lineamientos de uso de las presentaciones (10 páginas). |
| `Presentacion/` | Presentación corporativa completa de Nortica Systems (23 páginas). |
| `Service Report/` | Plantilla/ejemplo de reporte de servicio técnico. |
| `Technical Data/` | Hoja de datos técnicos de productos/equipos. |
| `Test/` | Material de prueba o borradores en proceso de validación. |

### `ui_presentacion/`
Contiene los elementos de identidad visual reutilizables en cualquier presentación o pieza gráfica.

| Carpeta | Contenido |
|---|---|
| `logos_nortica/` | Variantes del logotipo oficial de Nortica Systems para distintos fondos y usos (fondo oscuro en blanco, fondo oscuro blanco/cyan, versión por defecto, versión de manual de marca). |
| `imagen_fondo_ui/` | Imagen de fondo utilizada en interfaces o herramientas internas (`background_image.png`). |

### `firma_correos/`
Plantilla de firma de correo electrónico corporativa, lista para usar en clientes de correo (Outlook, Gmail, etc.).

| Archivo/Carpeta | Contenido |
|---|---|
| `firma.html` | Código HTML de la firma de correo corporativa. |
| `images/img_signature.png` | Imagen de firma incluida en la plantilla. |
| `images/logo_default.png` | Logo de Nortica Systems embebido en la firma. |

## Uso

Estos recursos están pensados como repositorio centralizado de referencia para la elaboración de presentaciones, propuestas comerciales, reportes técnicos y demás materiales de Nortica Systems, manteniendo consistencia de marca e información actualizada.
